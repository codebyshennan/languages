Behaviour

  • Press / anywhere on a language page → modal opens, input auto-focused
  • Type any word → press Enter → Web Speech API speaks it immediately
  • Lang auto-detected: /viet → vi-VN, /bahasa → id-ID, /spanish → es-ES
  • Last 5 spoken words shown as chips — click to re-speak
  • Press ESC or click backdrop to close
  • All logic in shared.js + shared.css — zero new pages/routes
